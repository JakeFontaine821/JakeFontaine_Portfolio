Veggie Slam

Veggie Slam is a 2D side scrolling game, the goal is to traverse the levels in order to complete the given objective. Instead of fighting off enemies in this platformer, you must swan dive and body slam the ground to knock them away. The idea of the game is fun and has a couple different possiblities outside of just reaching a goal. One of which I was able to implement in the time span was bouncing an enemy into a goal across the map.

Play on itch.io

Veggie Slam was made as my Associate's degree capstone project.

I programmed the whole game and designed the levels. I also created all enemy and player art.

The tilemap used for the map, all sound effects, and music were found on the Unity Asset Store.

Game Engine: Unity 2020.3.14

IDE used: Visual Studio 2019, Programmed in Unity C#

Time Frame: 8 Weeks

Python Computer Vision

My hand tracking program utilized the Mediapipe Python library to track hand landmarks on the screen. By tracking the landmark locations I was able to detect different hand symbols. Since i don't know ASL, I had some fun and tracked hand symbols from the anime Naruto.

My color detection program used screen capture to detect specific colors in a specified bounding box. Then perform the necessary post processing effects to ensure it's detecting the correct objects.

This project was done as a Mid-Term project.

Python libraries Used:
OpenCV, Mediapipe, MSS, Time, Numpy, Mouse

IDE used: Jupyter Notebook & Visual Studio Code, Programmed in Python

Time Frame: 2 weeks

Genetic Algorithm

My genetic algorithm is a trajectory guesser. The gene of the algorithm contains a height of launch and horizontal velocity in order to calculate a shot from a cannon that will hit a target below. Gravity is of course used for the equation however i did ignore wind resistance. The algorithm does get stuck from time to time so changing around range of variables creates intesting combinations of height and velocity.

The project was part of my final project for an Artificial Intelligence class.

All code is my own.

IDE used: Eclipse IDE, Programmed in Java

Time Frame: 1 Week
